# Environment

Activate Virtual Environemnt

```
# Create venv
virtualenv --python=3.10.13 .venv 

# Activate it
source .venv/bin/activate

# Deactivate it
deactivate
```

# Dependencies

Installing dependencies

```
pip3 install -r requirements.txt
```
